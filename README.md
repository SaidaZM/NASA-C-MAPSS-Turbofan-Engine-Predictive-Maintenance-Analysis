# NASA-C-MAPSS-Turbofan-Engine-Predictive-Maintenance-Analysis
A comprehensive FMDS analysis and predictive maintenance system for NASA C-MAPSS turbofan engines. Combines Weibull reliability modeling, survival analysis, and ML (Random Forest, SVM) to predict Remaining Useful Life. Features automated preprocessing, Kaplan-Meier estimation, and multi-class health classification.
